Converts a given string to lower case.
{% highlight php %}
<?php
lower(string $value)
{% endhighlight %}

* **value**: string to convert

## Example
{% highlight smarty %}
{lower('ThiS IS a STRING')}
{% endhighlight %}

## Output
{% highlight text %}
this is a string
{% endhighlight %}
