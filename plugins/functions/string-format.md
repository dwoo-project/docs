Formats a string using the sprintf function
<div class="code-box">
{% highlight php %}
<?php
string_format(string $value, string $format)
{% endhighlight %}
</div>

* **value**: the string to format
* **format**: the format to use, see [sprintf()](http://php.net/sprintf){:target="_blank"} for details
